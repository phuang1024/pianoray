CLI
===

Driver CLI manual.

``usage: pianoray [-h] [-V] [-v] -p PATHS -o OUTPUT graph``

- ``-h``: Help
- ``-V``: Version
- ``-p``: Path(s) to kernels e.g. ``build/kernels``. Pass the folder
  containing the folders of kernels.
- ``-o``: Output video file.
- ``graph``: Pipeline file and class name e.g. ``/path/file.py:PipelineClass``
