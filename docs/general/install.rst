Installation
============

Dependencies
------------

- Python3.8 or higher.
- FFmpeg.
- C++ compiler (``g++``).
- Python packages listed in ``requirements.txt``

Latest
------

PianoRay is distributed on PyPI. Install with

.. code-block:: sh

   pip install pianoray

Master Branch
-------------

May be unstable or have bugs.

.. code-block:: sh

   pip install git+https://github.com/phuang1024/pianoray
